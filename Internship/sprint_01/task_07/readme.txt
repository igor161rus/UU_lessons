Расширенный анализ данных


Цель:
Реализовать функцию для расчёта и отображения дополнительных статистических индикаторов,
например, стандартного отклонения цены закрытия.

Реализация:
Добавить соответствующие расчёты в модуль обработки данных и обновить визуализацию для отображения этих индикаторов.

Результат:
В модуле calculation добавлена функция (50) calculation_std вычислящая скользящее стандартное отклонение в окне из 5ти значений

Другие изменения:
В модуле main (21) добавлена проверка ввода значения threshold
