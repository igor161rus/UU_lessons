Реализовать функционал: Изменение размера холста

Ваша задача:
Добавление параметра для выбора стиля графика

Реализация:
Для изменения размера холста добавьте кнопку, которая открывает диалоговое окно (tk.simpledialog.askinteger) для ввода новых размеров.
После получения новых размеров обновите свойства width и height холста и создайте новый объект Image с новыми размерами.
