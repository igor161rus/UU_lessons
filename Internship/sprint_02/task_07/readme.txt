Реализовать функционал: Инструмент "Текст" для добавления текста на изображение

Ваша задача:
Реализовать функционал: Инструмент "Текст" для добавления текста на изображение

Цель:
?? Реализовать функцию для расчёта и отображения дополнительных статистических индикаторов, например,
стандартного отклонения цены закрытия.

Реализация:
Реализуйте инструмент "Текст", добавив кнопку "Текст" и открывая диалоговое окно
(tk.simpledialog.askstring) для ввода текста пользователем.
После ввода текста и выбора места на холсте кликом левой кнопки мыши используйте
ImageDraw.text((x, y), text, fill=self.pen_color) для добавления текста на изображение.

Добавление функции изменения цвета фона холста
Добавьте в интерфейс кнопку "Изменить фон", которая вызывает диалоговое окно для выбора цвета (colorchooser.askcolor()).
В обработчике этой кнопки получите выбранный пользователем цвет и примените его к фону холста,
используя метод self.canvas.config(background=new_color).
Сохранять при этом имеющийся рисунок не обязательно.
