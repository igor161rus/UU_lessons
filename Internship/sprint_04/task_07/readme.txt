Задача №7. Статистика по упражнениям - отображение статистики по выполненным упражнениям
Реализовать функционал: Подбрасывание монетки

Ваша задача:
Функционал: Симулирует подбрасывание монетки, сообщая пользователю результат ("Орел" или "Решка").

Реализация:
Реализация: При получении команды (Flip a Coin) бот случайным образом выбирает результат из двух возможных ("Heads", "Tails") и сообщает его пользователю.
