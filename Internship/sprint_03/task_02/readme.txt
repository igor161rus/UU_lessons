Задача №2. Инверсия цветов изображения
Реализовать функционал: Инверсия цветов изображения

Ваша задача:
Функционал: Меняет цвета изображения на противоположные, тем самым создавая "негатив" изображения.


Реализация:
Используется функция invert_colors, которая применяет ImageOps.invert из PIL (Python Imaging Library) к изображению.

__________________________________________
Результат:
    Добавлена функция invert_colors (73) Инвертирует цвета полученного изображения.
    В функции pixelate_and_send (203) добавлена проверка уровеня пользователя и выбор соответствующей обработки изображения:
        3 - пикселизация, 4 - инвертирование


Другие изменения:
    Переписана функция callback_query - организует соответствующие пункты меню, устанавливает уровеь пользователя
    Переписана функция get_options_keyboard - генерирует различные InlineKeyboards в зависимости от уровня пользователя,
        хранящегося в user_states.
    Изменена функция handle_message - функция для обработки входящих сообщений от пользователя.
