Задача №1. Изменение набора символов для ASCII арта из изображений
Реализовать функционал: Изменение набора символов для ASCII арта из изображений

Сейчас ASCII арт делается по умолчанию из '@%#*+=-:. ', @ - для самых тёмных участков, пробел для самых светлых.
Добавить возможность это менять


Ваша задача:
Функционал: пользователь может задавать свой набор символов для создания ASCII-арта из отправленного изображения.

Реализация:
Реализация: после получения изображения, бот запрашивает у пользователя набор символов,
 который будет использоваться для преобразования изображения в ASCII-арт.
 Это позволяет пользователям персонализировать итоговый результат.
