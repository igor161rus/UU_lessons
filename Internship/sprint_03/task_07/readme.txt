Задача №7. Случайный комплимент
Реализовать функционал: Случайный комплимент

Ваша задача:
Функционал: Отправляет пользователю случайный комплимент из предопределенного списка.

Реализация:
Реализация: При получении команды (Random Compliment) бот выбирает случайный комплимент из списка COMPLIMENTS и
отправляет его пользователю.
