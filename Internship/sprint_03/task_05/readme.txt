Задача №5. Изменение размера изображения для стикера
Реализовать функционал: Изменение размера изображения для стикера

Ваша задача:
Функционал: Адаптирует размер изображения для использования в качестве стикера в Telegram, ограничивая максимальный размер.

Реализация:
Функция resize_for_sticker изменяет размер изображения, сохраняя пропорции, чтобы его максимальное измерение не
превышало заданного максимума (например, 512 пикселей).
