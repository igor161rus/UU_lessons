Задача №3. Отражение изображения
Реализовать функционал: Отражение изображения

Ваша задача:
Функционал: Создает отраженную копию изображения по горизонтали или вертикали.

Реализация:
Функция mirror_image применяет метод transpose из PIL с параметрами Image.FLIP_LEFT_RIGHT или Image.
FLIP_TOP_BOTTOM для горизонтального или вертикального отражения соответственно.
